Project Overview

Theme: Pirates 🏴‍☠️

This project combines 3D modeling in Maya and immersive interaction in Unity. I designed and built a pirate-themed environment, featuring self-made 3D props and real-time interaction.

🧩 Tools Used

  • 💻 Autodesk Maya – for modeling & texturing assets
  • 🎮 Unity – for world building and interactivity

🎯 Goals

  • Practice creating optimized 3D models
  • Design interactive 3D experiences for users
  • Learn the Unity workflow for animation and gameplay

Snowman Winter Game

Two levels • Easy (collect & dodge) • Hard (reflect falling snowballs)

A winter-themed web game with two difficulties. In Easy, the snowman collects snow while dodging growing trees. In Hard, you control a bouncing bar to reflect falling snowballs. Sound effects keep the experience lively.

Technologies Used

  • p5.js (creative coding)
  • Object-Oriented Programming (classes for Snowman & game objects)
  • VS Code & GitHub Pages

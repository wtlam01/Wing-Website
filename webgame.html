❄️ Web-Based Interactive Audio-Visual Game

A winter-themed game featuring two levels: In the easy mode, the snowman collects snow while dodging growing trees; in hard mode, a bouncing bar must reflect falling snowballs. Fun sound effects inspired by Mario enhance the joyful and challenging gameplay.

🔧 Technologies Used

  • 🧊 p5.js – for creative coding
  • 🧠 Object-Oriented Programming (Classes: Snowman, GameObject...)
  • 💻 Visual Studio Code, GitHub Pages
